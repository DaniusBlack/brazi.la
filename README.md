====

A fresh new CMS made with MEAN and love :P

## Installing

Run this:

```bash
git clone git@github.com:NodeJS-Brasil/brazi.la.git brazila
cd brazila
npm install
```

Create the config.js with your environment configs.
Use the config-exemple.js for instructions
